# 🚀 ADVANCE - CORE
### Sistema de Conexión de Activos de Alto Perfil

Plataforma inteligente diseñada para conectar compradores y vendedores de alto valor (Autos y Bienes Raíces) mediante telemetría e IA.

## 🛠️ Tecnologías
- **Backend:** Python (Flask)
- **Frontend:** HTML5, JS, CSS3
- **Base de Datos:** Supabase (PostgreSQL)
- **Despliegue:** Vercel
